# משחק השליח — Go-Order

## 🎯 מה זה?
משחק ארקייד דו־ממדי (2D) לשחקן יחיד: אתם שליחים בעיר מצוירת, שמבצעים משלוחים נגד השעון בין רחובות וגשרים — תוך הימנעות ממכשולים דינמיים, צבירת מטבעות ושדרוג כלי רכב כדי להתקדם בשלבים.

---

## ⚙️ מאפייני משחק בקצרה
- **פלטפורמה:** מחשב ונייד (מובייל)  
- **סגנון גרפי:** דו־ממדי (2D) מצויר  
- **קהל יעד:** ילדים ונוער  
- **סוג משחק:** שחקן יחיד (Single Player)

---

## 🎮 רכיבים רשמיים

### 1) שחקנים
- מותאם לילדים ונוער שאוהבים משימות וניווט.  
- חוויה לשחקן יחיד; ניתן להוסיף דירוגים אסינכרוניים (Leaderboards).

### 2) יעדים
- **מטרה בכל שלב:** להגיע לסכום מטבעות הנדרש.  
- **ניצחון:** השלמת כל השלבים.  
- **כישלון:** אי־עמידה בזמן/במשלוח.  
- **הצגה:** טיימר, מונה משלוחים ומיני־מפה על המסך.

### 3) תהליכים (Game Loop)
- **התחלה:** מקבלים אופניים ומשלוח ראשון; מדריך קצר; HUD עם זמן/כסף/מפה/משלוח פעיל.  
- **הליבה (Core Loop):**  
  1. קבלת משלוח →  
  2. ניווט והימנעות ממכשולים →  
  3. מסירה ליעד →  
  4. קבלת מטבעות ובונוסים →  
  5. שדרוגים בין שלבים.  
- **סיום:** עומדים ביעד הזמן והמשלוחים כדי להתקדם; אין “הפסד מוחלט” — מנסים שוב.

### 4) חוקים
- כל משלוח מוגבל בזמן.  
- התנגשות במכשולים (למשל שמן, שוטרים, מחסומים) גורמת לעיכוב.  
- פריטים מיוחדים מוסיפים זמן/מהירות.  
- שדרוגים תלויים בכסף שצברתם; המדריך מסביר השפעות פריטים.

### 5) משאבים
- **כסף (מטבעות):** לשדרוג כלי רכב ורכישת פריטים(skin).  
- **זמן:** קריטי להשלמת משלוחים.  
- **פריטים:** בוסטרים(תוספת זמן, מהירות) שמן, מחסומים ועוד.  
- **איך משיגים:** משלוחים, בונוסים ופריטים על המפה.  
- **איזון/נדירות:** בתחילה זמן ארוך ורווח נמוך וככל שמתקדמים הזמן מתקצר העומס גובר והתגמול בהתאם.  
- **הצגה:** מיני־מפה, טיימר, מונה כסף ב־HUD.

### 6) עימותים
- **מול המערכת:** עומסי תנועה, מזג־אוויר, הפגנות, שוטרים ושמן על הכביש.

### 7) גבולות העולם
- **סוג עולם:** עיר פתוחה.  
- **גבולות:** ללא גבולות קשיחים; המבנים/כבישים משרטטים את המרחב.  
- **ניווט:** מיני־מפה וחיצים.  
- **עקרונות תכנון:** משמעות (מסחר/מגורים/תעשייה), ניידות (נתיבים ברורים), עניין (אירועים דינמיים), הכוונה (שילוט/סמלים).
